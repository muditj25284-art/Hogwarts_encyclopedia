Hogwarts Encyclopedia
A web app to explore Harry Potter characters. You can search for characters, filter them by house, and sort them by name.

API
HP-API - https://hp-api.onrender.com/api/characters
No API key needed. Returns a list of characters with name, house, species, role, and image.

Features
Display all characters fetched from the API
Search characters by name
Filter characters by Hogwarts house
Sort characters A to Z or Z to A

Technologies Used
HTML
CSS
JavaScript

How to Run
Download or clone the repo and open index.html in a browser.
