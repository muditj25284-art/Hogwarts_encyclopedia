Hogwarts Encyclopedia
A web app to explore Harry Potter characters built as part of a frontend web development assignment at Rishihood University.

Live Demo
https://muditj25284-art.github.io/Hogwarts_encyclopedia/

API
HP-API — https://hp-api.onrender.com/api/characters  
No API key needed. Returns characters with fields like name, house, species, and image.

Features
* Fetches and displays all characters from the API
* Search characters by name
* Filter characters by Hogwarts house
* Sort characters A to Z or Z to A
* Shows a count of how many characters match the current filters
* Shows a "no results" message when filters return nothing

Array Methods Used
.filter() — used for name search and house filtering
.sort() — used for A to Z and Z to A sorting
.map() — used to render character cards
